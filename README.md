# Dermcare Frontend

Ứng dụng frontend cho hệ thống phòng khám da liễu trực tuyến **Dermcare**, được xây dựng bằng **Next.js (App Router)**, **TypeScript**, **Tailwind CSS** và bộ component theo phong cách **shadcn/ui**.

## Công nghệ chính

- **Next.js** (App Router) với TypeScript.
- **Tailwind CSS** với custom theme màu trắng và xanh y tế.
- **UI primitives** theo phong cách shadcn/ui (button, layout, card...).

## Cấu trúc thư mục

- `src/app` – Routing & layout theo App Router.
- `src/components` – Các UI components dùng chung (theo phong cách shadcn/ui).
- `src/features` – Các module nghiệp vụ (đặt lịch, hồ sơ bệnh án, bác sĩ...).
- `src/lib` – Helpers, utils.
- `src/hooks` – Custom React hooks.
- `src/types` – Kiểu dữ liệu dùng chung.
- `src/config` – Cấu hình tĩnh (siteConfig, constants...).

## Chạy dự án

```bash
npm install
npm run dev
```
